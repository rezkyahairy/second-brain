# Date Helper

## Get Current Date
```bash
date +"%Y-%m-%d"
```

## Get Current Time
```bash
date +"%H:%M"
```

## DATE_CHECK
**ALWAYS verify the year matches internet time before creating notes.**
System clocks can be wrong. When in doubt, check against a known source.

## Template Variables
- `{{date}}` → YYYY-MM-DD
- `{{time}}` → HH:MM
- `{{datetime}}` → YYYY-MM-DDTHH:MM
